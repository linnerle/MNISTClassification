\section{Introduction}

\subsection{Background and Motivation}
MNIST digit classification is a common way to test how neural networks learn. Since this was an individual project, I used it as a chance to understand that learning process myself. The assignment required manual implementation, so instead of using high-level deep-learning layers, I implemented forward propagation, loss calculation, and backpropagation directly using NumPy.

\subsection{Problem Definition}
The main task is to classify $28\times 28$ grayscale images of digits (0--9). In this report, I compare the two required architectures based on how they train, how well they perform on test data, and where they still make mistakes.

\subsection{System Overview}
I study two manually implemented models:
\begin{enumerate}
    \item \textbf{Multilayer Perceptron (MLP):} a 2-layer fully connected network with Sigmoid in the hidden layer and Softmax in the output layer, trained for up to 30 epochs.
    \item \textbf{Convolutional Neural Network (CNN):} one valid-mode convolution layer with ReLU, followed by a fully connected Softmax output layer, trained for up to 5 epochs.
\end{enumerate}
For both models, I use the same MNIST train/test split and evaluate them using training logs and the figures included in this report.
