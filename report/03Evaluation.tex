\section{Evaluation}

\subsection{Experimental Settings}
Experiments follow the exact training loops in the template files. MLP is trained with hidden size $128$, learning rate $0.5$, batch size $128$, and $30$ epochs; CNN uses one $7\times 7$ valid convolution kernel, learning rate $0.05$, batch size $64$, and $5$ epochs.

\subsection{Training Progression and Feature Extraction}
From Figure \ref{fig:mlp_curves}, I see that the MLP improves quickly at the start and then keeps improving more slowly across later epochs. In this run, test accuracy rises from $0.9166$ (epoch 1) to $0.9790$ (epoch 30).

Figure \ref{fig:cnn_curves} shows a different behavior for CNN. Training loss goes down in all 5 epochs, but test accuracy goes up and down, with a dip at epoch 4 ($0.8803$) before ending at $0.9178$. To me, this suggests unstable short-run behavior rather than steady generalization from epoch to epoch.

In Figure \ref{fig:mlp_weights_grid}, different hidden units appear to focus on different stroke patterns. Some emphasize vertical or horizontal strokes, while others capture curved or crossing parts of digits. This suggests the hidden layer learns useful parts that the output layer later combines for final class decisions.

\subsection{Qualitative Profiling and Confusion Analysis}
Figure \ref{fig:mlp_cm} and Figure \ref{fig:cnn_cm} show a clear difference in error patterns. The MLP matrix is more concentrated on the diagonal, while the CNN matrix has more spread into off-diagonal cells after 5 epochs. For example, MLP mistakes like $4\rightarrow 9$ (15) and $5\rightarrow 3$ (13) are limited, but CNN mistakes like $8\rightarrow 3$ (132), $9\rightarrow 3$ (122), and $0\rightarrow 2$ (101) are much larger. I read this as CNN still being under-trained at this budget, especially on visually similar digits.

A key limitation is that the training budgets are not equal (30 MLP epochs versus 5 CNN epochs, following assignment rules). Because of that, the performance gap should be read as a comparison under required settings, not as a final statement about each model's full potential.
